# Ejercicios del Libro de Rust

Este repositorio contiene mi progreso y los ejercicios prácticos realizados siguiendo el Libro de Rust en castellano.

## Propósito

El objetivo de este proyecto es documentar mi aprendizaje del lenguaje de programación Rust, implementando los ejemplos y ejercicios propuestos en el libro oficial traducido. Aquí encontrarás el código fuente organizado por capítulos.

## Estructura del repositorio

El código está organizado siguiendo la estructura de los capítulos del libro:

- `1.2-hola_mundo/`: Primeros pasos con Rust
- `capitulo-X/`: Ejercicios correspondientes a los diferentes temas tratados

## Cómo utilizar este repositorio

Puedes navegar por las carpetas para revisar las implementaciones de cada ejercicio. Si deseas ejecutar cualquiera de los proyectos, asegúrate de tener instalado Rust en tu sistema:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/elarreglador/libro_rust.git
   ```

2. Navega al directorio del ejercicio que te interese:
   ```bash
   cd libro_rust/nombre-del-ejercicio
   ```

3. Compila y ejecuta:
   ```bash
   cargo run
   ```
